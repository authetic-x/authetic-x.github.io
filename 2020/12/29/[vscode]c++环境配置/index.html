vscode报错“找不到C++头文件”的解决办法

发表于 2020-12-29 分类于 错误日志 ， C++

一.确认基本环境已安装

要使用 Vscode 运行 C++ 程序你至少要做以下几件事情(这里默认在windows10的环境下)：

  1. 安装 MinGW 编译器，Cygwin 也可以，不会装可以搜下教程
  2. 配置 MinGW 环境变量，将 MinGW 安装目录下 bin 文件夹的路径配置到环境变量中
  3. vscode 安装 C/C++ 扩展

一般来说正确的完成了以上三步就可以跑 C++ 的程序了，以前是没问题的，但我的 vscode 今天突然抽风了。
